# Security Policy

## Supported Versions

The following versions of this project are currently supported with security updates.

| Version | Supported |
| ------- | --------- |
| 5.1.x   | ✅ |
| 5.0.x   | ❌ |
| 4.0.x   | ✅ |
| < 4.0   | ❌ |

## Reporting a Vulnerability

The security of this project is taken seriously. If you believe you have found a vulnerability, please report it privately.

### Reporting Process

To report a security issue, send an email to **kha5133@psu.edu** with the subject line:

**[Security Vulnerability] Project Name**

Please include:

- A description of the vulnerability
- Instructions to reproduce the issue
- The potential impact
- Affected versions, branches, or environments
- Any suggested mitigation or fix, if available

### Response Timeline

We aim to:

- Acknowledge receipt within **3 business days**
- Provide a status update within **7 business days**
- Resolve confirmed issues as quickly as possible based on severity and complexity

### Coordinated Disclosure

Please avoid public disclosure until the issue has been investigated and a fix or mitigation has been released.

We appreciate your help in improving the security of this project.
